# 📱 Mini Social Media App

A simple social media web application where users can create posts, comment, and interact with other users.
This project demonstrates basic social media features and full-stack web development concepts.

---

## 🚀 Features

* 👤 User profile
* 📝 Create and view posts
* 💬 Comment system
* ❤️ Like
* 📱 Responsive user interface

---

## 💻 Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB

---

## 📂 Project Purpose

This project was built to practice web development skills and understand how social media platforms work.
